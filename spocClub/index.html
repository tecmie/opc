Secondary School Practical Open-source Curriculum

Lessons

Lessons in the SPOC curriculum are currently majoring in the technical and soft skills aspects of learning. The learning experience is also more of a peer to peer learning experience where a teacher how knows how to code is not necessarily required for te club to run. The students practice soft skills such as leadership and group work. The technical skills will be discussed more in depth when we get to the specifics of what programming languages are learnt in SPOC.

Student Board

The SPOC program being autonomous, has a group of three student representative that take up the leadership in the student board.These roles include, the president, the vice president and the secretary.

High Schools

SPOC is currently launched in about 9 schools in Nairobi.

The code

We learn two main programming languages in the first level of SPOC. These are HTML and CSS. HTML stands for Hyper Text Markup Language and CSS which stands for Cascading Style Sheets.

Activities

During a SPOC class, students learn how to learn by working in pairs and also in groups.

Moringa School, is where it all started. SPOC is a project within Moringa School. Kinda like a mother to SPOC. Moringa School is a coding boot camp that teaches students how to become amazing junior and mid-level software developers in the African market.
